# **App Name**: MediScribe

## Core Features:

- Real-time Transcription: Transcribes spoken medical notes into text in real time.
- Offline Functionality: Allows for transcription even without an internet connection.
- Medical Terminology Tool: Generative AI powered tool that learns and accurately interprets complex medical terms and jargon for precise transcription of the nuances of medical dictation, and identifies key terms in the record.
- Microsoft Word Add-in: Integrates as an add-in for Microsoft Word for direct dictation and formatting.
- Standalone Application: Functions as a standalone application with full transcription and editing capabilities.
- Secure Local Storage: Securely stores transcribed records locally on the device, for cases when immediate cloud backup is not an option.
- Customizable Templates: Offers customizable templates for various medical reports and documentation.
- Medical Auto-Correct: Generative AI powered tool that corrects medical word typo errors or misspelled words using an in-built machine learning model.

## Style Guidelines:

- Primary color: Deep blue (#29434E) to evoke trust and professionalism.
- Background color: Light gray (#E8EAED), desaturated to provide comfortable contrast and readability.
- Accent color: Teal (#3E92A7) to highlight important UI elements without being distracting.
- Body and headline font: 'Inter', sans-serif, for a modern, clean, and easily readable interface.
- Use clear and professional icons representing common medical terms and actions.
- Clean and intuitive layout to ensure ease of use for medical professionals.
- Subtle transitions to provide a smooth and responsive user experience.
