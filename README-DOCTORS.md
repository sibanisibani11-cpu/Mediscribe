# MediScribe - AI Medical Transcription for Doctors

**Professional offline medical transcription powered by OpenAI Whisper AI**

## 🏥 For Healthcare Professionals

MediScribe is designed specifically for doctors and medical professionals who need accurate, fast, and secure medical transcription. Everything runs locally on your computer - no internet required after installation.

## ✨ Key Features

### 🎤 **Voice Recording**
- **Real-time transcription** as you speak
- **Medical terminology optimization** with AI auto-correction
- **Hands-free operation** with global keyboard shortcuts
- **Audio feedback** with start/stop beeps

### 📁 **File Transcription**
- **Upload audio files** from recordings, voice memos, or dictation devices
- **Batch processing** for multiple files
- **Timeline segments** showing when each part was spoken
- **Support for all audio formats** (MP3, WAV, M4A, etc.)

### 💻 **Direct Integration**
- **Type directly into Word** or any application
- **Global shortcuts**: `Ctrl+Shift+M` (show/hide app), `Ctrl+Shift+R` (quick record)
- **System tray integration** - always available when needed
- **Save transcriptions** as text files or Word documents

### 🔒 **Privacy & Security**
- **100% offline** - no data sent to servers
- **HIPAA compliant** - all processing on your local machine
- **Secure AI models** bundled with the app
- **No internet dependency** for transcription

## 🚀 Getting Started

### Installation
1. **Download** the MediScribe installer for your platform:
   - Windows: `MediScribe-Setup-1.0.0.exe`
   - macOS: `MediScribe-1.0.0.dmg`
   - Linux: `MediScribe-1.0.0.AppImage`

2. **Run the installer** - everything is included (AI models bundled)

3. **Launch MediScribe** from your applications or system tray

### First Use
1. **Grant microphone permissions** when prompted
2. **Test voice recording** with a short phrase
3. **Enable "Direct Typing"** to type into Word automatically
4. **Try offline transcription** by uploading an audio file

## 🎯 Usage Guide

### Voice Recording Mode
1. Click the **microphone button** or press `Ctrl+Shift+R`
2. **Speak clearly** - the AI will transcribe in real-time
3. **Click stop** when finished
4. **AI auto-correction** will improve medical terminology
5. **Text appears** in MediScribe and optionally in your target app

### File Transcription Mode
1. Switch to **"Offline Files" tab**
2. **Drag and drop** audio files or click to select
3. **Wait for processing** - progress bar shows status
4. **Review transcription** with timeline segments
5. **Copy or save** the results

### Keyboard Shortcuts
- `Ctrl+Shift+M` - **Show/hide** MediScribe
- `Ctrl+Shift+R` - **Quick record** toggle
- `Ctrl+C` - **Copy** transcription to clipboard
- `Ctrl+S` - **Save** transcription to file

## 💡 Tips for Best Results

### Speaking Techniques
- **Speak clearly** at normal pace
- **Pause briefly** between sentences
- **State punctuation** when needed ("period", "comma")
- **Spell difficult terms** if needed

### Audio File Quality
- **Use good quality recordings** (clear audio)
- **Minimize background noise**
- **Single speaker works best**
- **Standard formats**: WAV, MP3, M4A recommended

### Medical Terminology
- The AI is **trained on medical language**
- **Drug names** are recognized accurately
- **Anatomical terms** are handled correctly
- **Procedure names** are transcribed properly

## 🔧 Troubleshooting

### Common Issues

**"Microphone not working"**
- Check system microphone permissions
- Try a different microphone or headset
- Restart MediScribe

**"Direct typing not working"**
- Make sure target application (Word) is focused
- Check if Direct Typing is enabled in settings
- Try copying and pasting instead

**"Offline transcription slow"**
- Large files take more time to process
- Close other applications for better performance
- Consider breaking long recordings into smaller files

**"App won't start"**
- Run as administrator (Windows) or check permissions (Mac)
- Make sure your system meets requirements
- Contact support if issue persists

### System Requirements
- **Windows**: Windows 10 or later
- **macOS**: macOS 10.15 or later
- **Linux**: Ubuntu 18.04 or equivalent
- **RAM**: 4GB minimum, 8GB recommended
- **Storage**: 3GB free space (includes AI models)
