# 🚀 MediScribe v1.0.4 - Official Stable Release

Welcome to the official stable release of **MediScribe v1.0.4**. This version introduces the premium "Midnight Cobalt" theme, enhanced 3-stage AI verification, and stable builds for all major platforms.

## 📦 Download Guide (Which file do I need?)

Please choose the installer that matches your operating system:

*   **Windows Users** → Download `MediScribe.Setup.1.0.4.exe` (Works on Windows 10 & 11)
*   **Mac Apple Silicon (M1/M2/M3/M4)** → Download `MediScribe-1.0.4-arm64.dmg`
*   **Mac Intel Users (2006-2020)** → Download `MediScribe-1.0.4.dmg`
*   **Linux Users** → Download `MediScribe-1.0.4.AppImage`

---

## 🛠️ Installation Instructions

### Windows
1. Download **MediScribe.Setup.1.0.4.exe**.
2. Double-click the file to begin installation.
3. Once completed, you can launch MediScribe from your Desktop shortcut.

### macOS
1. Download the `.dmg` file corresponding to your Mac's processor.
2. Open the downloaded file and drag the **MediScribe** icon into your **Applications** folder.
3. Launch MediScribe from the Applications folder.
   * *Tip: If you see a security warning, right-click the app and choose "Open".*
